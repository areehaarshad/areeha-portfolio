## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4
- Framer Motion (animations)
- Lucide React (icons)

## Running Locally

```bash
npm install
npm run dev
```

## Building for Production

```bash
npm run build
```

Output is generated in the `dist/` folder, ready to deploy to any static host (GitHub Pages, Netlify, Vercel).

## Deployment

This site is deployed via **GitHub Pages**. The build uses a relative base path (`base: './'` in `vite.config.ts`) so it works correctly whether hosted at the repository root or in a subdirectory.
